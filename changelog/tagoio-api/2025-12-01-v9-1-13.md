## Highlights

- Install templates from any region (US, EU, etc.) into accounts on different regions
- Use the `host` parameter to specify the source region's API endpoint

**Example Usage:**

Installing a US template in EU region:

```
GET https://admin.eu-w1.tago.io/template/{template_id}?host=api.tago.io
```
