# Specification

The TagoTiP protocol specifications are maintained on GitHub. They cover the complete protocol grammar, parsing rules, ABNF, frame structure, and test vectors.

## TagoTiP Specification

The core protocol specification covering frame format, methods (PUSH, PULL, PING), operators, suffixes, sequence counters, response codes, and transport-specific behavior.

[Read the TagoTiP specification on GitHub](https://github.com/tago-io/tagotip/blob/main/TagoTiP.md) | [Full specification](./tagotip-specification)

## TagoTiP/S Specification

The encrypted communication specification covering the AEAD envelope format, cipher suites, key derivation, nonce construction, flags byte, and test vectors.

[Read the TagoTiP/S specification on GitHub](https://github.com/tago-io/tagotip/blob/main/TagoTiPs.md) | [Full specification](./tagotips-specification)
