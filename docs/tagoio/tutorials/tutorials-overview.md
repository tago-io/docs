## Tutorial Categories

### IoT Protocols
- **MQTT**: Message queuing telemetry transport implementation
- **LoRaWAN**: Long-range, low-power wireless communication
- **Sigfox**: Ultra-narrowband IoT connectivity

### Hardware Platforms
- **Single-board computers**: Raspberry Pi, Beagle Bone Black
- **Development kits**: Thinxtra Xkit, LoRa-E5 Module
- **Commercial sensors**: Adeunis, Sens'it, RadioBridge

### Applications
- **Environmental monitoring**: Temperature, humidity, air quality
- **Asset tracking**: GPS location, movement detection
- **Smart home**: Power monitoring, automation
- **Automotive**: Vehicle telemetry, fleet management

### Development Tools
- **Custom widgets**: ECharts integration, visualization
- **Report generation**: PDF services, data export
- **Documentation**: Markdown formatting, technical writing
